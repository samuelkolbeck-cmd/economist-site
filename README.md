# Samuel Kolbeck - Digital CV

A professional digital portfolio and CV built with React (Frontend) and Node.js (Backend).

## Features
- Professional Timeline for Experience & Education
- Minimalist Academic Design
- Light/Dark Mode Toggle
- Contact Form Integration

## Tech Stack
- **Frontend:** React, TypeScript, Vite, Vanilla CSS
- **Backend:** Node.js, Express, TypeScript
- **Deployment:** Render / GitHub
